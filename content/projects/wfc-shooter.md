A top-down shooter where every level is procedurally generated using the Wave Function Collapse algorithm. The WFC system assembles stone structures, pathways, and environmental features on desert terrain with palm trees, creating unique layouts each run. Gameplay features enemies, projectiles, explosions, and a health system.
